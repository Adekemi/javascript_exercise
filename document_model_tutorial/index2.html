Selecting with jQuery

You can use the $ sign to select from the DOM! It will feel a lot like selecting using querySelectorAll()

For example:

  • Selecting all a tags
    • $(a)
  • Selecting all eleemts from the class 'container'
    • $(".container")
  • Selecting all elements with id "special"
    • $("#container")
